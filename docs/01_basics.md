# 第1章：シェルスクリプトの基本の「き」 🌱

## シェルスクリプトってなに？

普段ターミナルで `ls` とか `cd` とか打ってるよね？
あれを**ファイルにまとめて、上から順番に自動で実行してくれるメモ**みたいなのが「シェルスクリプト」だよ。

拡張子は `.sh` をつけるのが一般的！

## 1. 魔法の言葉 `#!/bin/bash`

スクリプトの1行目には、必ずこれを書く決まりがあるよ。
「これはBash（バッシュ）という言葉で書いてあるから、Bashで実行してね！」っていう意味。

```bash
#!/bin/bash
```

これを **シバン (Shebang)** って呼ぶよ。

## 2. 文字を表示してみよう (`echo`)

画面に文字を出すには `echo` (エコー) を使うよ。

```bash
#!/bin/bash

echo "こんにちは！"
echo "今日の日付は..."
date
```

### 実行してみよう！

1. `test.sh` というファイルを作って、上の内容をコピペして保存。
2. ターミナルで実行！

```bash
bash test.sh
```

「こんにちは！」って出た？成功！🎉

## 3. 変数（箱）を使おう

名前とか、長いパスとかを「箱」に入れておくと便利だよ。
箱に入れるときは `=` を使って、中身を使うときは `$` をつけるのがルール！

```bash
#!/bin/bash

# NAME という箱に "Kokko" を入れる（スペース空けちゃダメ！）
NAME="Kokko"
DOTFILES_DIR="/home/kokko/dotfiles"

# 中身を使うときは $ をつける
echo "ようこそ、$NAME さん！"
echo "設定ファイルは $DOTFILES_DIR にあるよ。"
```

### 💡 ポイント
- `NAME = "Kokko"` みたいに **= の前後にスペースを入れるとエラーになる** から注意！（これ初心者が一番ハマる罠！）
- 文字列の中に変数を埋め込むときは `"$NAME"` みたいにダブルクォートで囲むのが安全だよ。
