- 朝の分岐(重要): **7時より前に起きた日だけ「筋トレ→登校」。7時以降に起きた日は筋トレせず登校**。
  時間割はこの条件で組む(提案する起床時刻が7時より前なら筋トレを入れ、7時以降なら筋トレを入れない)。

  勉強は基本的に学校で行う。時間割に必ず登校ブロックを入れること。
- 身支度＋登校(移動)は**合わせて約30分**でよい(別々に30分ずつ取らない。1ブロックにまとめてよい)。
  ただし**雨天時は登校だけで約30分**かかるので多めに取る(※天気は現状自動取得していない。雨予報の
  日は手動で伝えてくれれば反映する)。

  「いつ・何分やるか」を明示する。時刻の根拠は時間割(proposed_schedule)の対応枠。
