### RAM 制約メモ

ローカル LLM を現サーバーで動かす場合の現実的な上限：
- **Phi-3 mini (3.8B)** Q4量子化：約 2.5 GB → 動作可
- **Llama 3.2 3B** Q4量子化：約 2.0 GB → 動作可
- **7B モデル** Q4量子化：約 4.5 GB + OS で限界 → 非推奨
- **13B 以上**：不可

Phase 1 では Gemini API を使い、ローカル推論は Phase 2 以降（GPU PC）に委ねる。

### ストレージレイアウト（予定）

```
/dev/sda2  116 GB  システム領域（/ マウント済み）
/dev/sdb   1.8 TB  データ領域
  └── /mnt/data/
        ├── photos/        # 写真・動画
        ├── documents/     # ドキュメント類
        ├── shared/        # 外部共有用（限定リンク対象）
        └── backup/        # バックアップ受け口
```
