# 10. バックアップ（restic）

## 概要

`restic`でローカル暗号化バックアップを行っています。バックアップ先は同一ディスク内の
`/mnt/data/backup/restic-repo`なので、**ディスク自体（`/dev/sdb`）が壊れた場合は失われます**。
3-2-1原則の「別媒体・オフサイト」はまだ満たしていません（今後の課題）。

## バックアップ対象

- `/mnt/data/photos`, `/mnt/data/documents`, `/mnt/data/immich`（写真・ドキュメント本体）
- `/mnt/data/ai`（n8nワークフロー、Butler Botの記憶）
- Immich PostgreSQLの`pg_dump`（`/mnt/data/backup/db-dumps/immich_latest.sql`）
- OpenWebUI・Uptime Kumaの設定（Dockerボリューム）
- `.env`（APIキー等の設定。**この中に含まれる`RESTIC_PASSWORD`を失うとバックアップ自体が復元不可能になる点に注意**）

対象外：`pgdata`ボリューム生データ（pg_dumpで代替）、`model-cache`（再ダウンロード可能）、
Dockerイメージ本体、`docker-compose.yml`等（git管理下のため）。

## 自動実行

毎日03:30に`systemd`タイマーで自動実行されます。

```bash
systemctl list-timers homeserver-backup.timer
sudo journalctl -u homeserver-backup.service --since today
```

初回セットアップ時（別環境に構築する場合）：

```bash
sudo cp scripts/systemd/*.service /etc/systemd/system/
sudo cp scripts/systemd/*.timer /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable --now homeserver-backup.timer
```

## 手動実行

```bash
cd ~/dotfiles/server
bash scripts/backup.sh
```

## スナップショット確認

```bash
cd ~/dotfiles/server
source .env
export RESTIC_PASSWORD RESTIC_REPOSITORY
restic snapshots
```

## 復元方法

**特定のファイルだけ復元**（例: 間違って消した写真）：

```bash
source .env && export RESTIC_PASSWORD RESTIC_REPOSITORY
restic snapshots                          # 復元したいスナップショットIDを確認
restic restore latest --target /tmp/restore-test --include /mnt/data/photos
```

**丸ごと復元**（ディスク交換後などフルリストア）：

```bash
restic restore latest --target /mnt/data-restored
```

**Immichデータベースの復元**：

```bash
# 1. restic restoreでdb-dumps/immich_latest.sqlを取り出す
# 2. Immichコンテナを止める
docker compose stop immich-server immich-microservices
# 3. リストア
docker exec -i immich_postgres psql -U immich -d immich < immich_latest.sql
# 4. 再起動
docker compose up -d immich-server immich-microservices
```

## トラブルシューティング

**`restic`コマンドで毎回パスワードを聞かれる**：`.env`から`RESTIC_PASSWORD`・`RESTIC_REPOSITORY`を
`export`し忘れている。上記の`source .env && export ...`を忘れずに。

**バックアップが失敗する / ディスク容量不足**：

```bash
df -h /mnt/data
restic stats
restic forget --keep-daily 7 --keep-weekly 4 --keep-monthly 6 --prune  # 古いスナップショットを整理
```
