# 06. n8n ワークフロー設定

## n8n とは

GUIでワークフローを組める自動化ツールです。
「毎朝8時にGoogleカレンダーを見てDiscordに通知する」といったことを
プログラミングなしで設定できます。

---

## ステップ1: n8n にログイン

http://kokko-server-pavilion:5678 にアクセス（Tailscale接続必須）

- ユーザー名: `kokko`
- パスワード: `.env` の `N8N_PASSWORD`

---

## ステップ2: Googleカレンダー連携の設定

### OAuth2認証の設定

1. 右上メニュー → **「Settings」** → **「Credentials」**
2. **「Add Credential」** → **「Google Calendar OAuth2 API」**
3. 以下の情報を入力（Google Cloud Consoleで取得）：

#### Google Cloud Console での設定

1. https://console.cloud.google.com にアクセス
2. プロジェクトを作成（例: homeserver）
3. **「APIとサービス」** → **「ライブラリ」** → 「Google Calendar API」を有効化
4. **「OAuth同意画面」** を設定：
   - ユーザーの種類: **外部**
   - アプリ名: homeserver
   - テストユーザーに自分のGmailを追加
5. **「認証情報」** → **「認証情報を作成」** → **「OAuthクライアントID」**
   - アプリケーション種類: **ウェブアプリケーション**
   - 承認済みリダイレクトURI: `http://kokko-server-pavilion:5678/rest/oauth2-credential/callback`
6. クライアントIDとシークレットをコピー

n8nの認証情報画面に入力 → **「Sign in with Google」** → Googleアカウントで認証

## ステップ3: 毎朝のデイリーブリーフィング ワークフロー

### 作成するワークフロー

```
[Schedule: 毎日8:00] → [Google Calendar: 今日の予定取得] → [Discord: 通知送信]
```

### 手順

1. n8nで **「New Workflow」** をクリック
2. **「+」** → **「Schedule Trigger」**
   - Trigger Rule: `Cron`
   - 設定: `0 8 * * *`（毎日8時）
3. **「+」** → **「Google Calendar」** ノードを追加
   - Operation: `Get Many`
   - Calendar: メインカレンダー
   - Time Min: `{{ $now.startOf('day').toISO() }}`
   - Time Max: `{{ $now.endOf('day').toISO() }}`
4. **「+」** → **「Code」** ノードで整形：
   ```javascript
   const events = $input.all();
   if (events.length === 0) {
     return [{ json: { message: "今日の予定はありません。" } }];
   }
   const list = events.map(e => {
     const start = e.json.start?.dateTime || e.json.start?.date;
     return `・${start} - ${e.json.summary}`;
   }).join('\n');
   return [{ json: { message: `📅 **今日の予定**\n${list}` } }];
   ```
5. **「+」** → **「Discord」** ノード
   - Webhook URL: Discord の `#butler-notify` チャンネルのWebhook URL
   - Content: `{{ $json.message }}`

### Discord Webhook URLの取得

1. `#butler-notify` チャンネルを右クリック → **「チャンネルの編集」**
2. **「連携サービス」** → **「ウェブフック」** → **「新しいウェブフック」**
3. URLをコピー

## ステップ4: ディスク残量アラート

```
[Schedule: 毎日12:00] → [Execute Command: df] → [IF: 残量 < 10%] → [Discord: アラート]
```

1. Schedule Trigger: `0 12 * * *`
2. **「Execute Command」** ノード:
   ```bash
   df /mnt/data --output=pcent | tail -1 | tr -d ' %'
   ```
3. **「IF」** ノード:
   - Value 1: `{{ $json.stdout }}`
   - Operation: `Larger`
   - Value 2: `90`
4. **「Discord」** ノード（TRUE側）:
   - Content: `⚠️ ストレージ残量が10%を切りました！`
